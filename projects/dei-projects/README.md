# DEI Analytics Projects
### By Lorenzo Di Salvatore
Work and Organizational Psychology | HR Data Analytics Specialist

This folder contains five specialized analytics projects focused on Diversity, Equity, and Inclusion (DEI) and related HR metrics. Each project applies evidence-based analysis to real-world datasets to provide actionable insights for HR professionals and organizational leaders.

## Project Overview

| Project | Focus Area | Key Insights |
|---------|------------|--------------|
| [DEI Dashboard EMEA](./dei-dashboard-emea/) | DEI Analytics (EMEA focus) | Gender representation, ethnic diversity, attrition disparities, gender pay gap, representation trends |
| [Human Resources Analytics](./human-resources-analytics/) | HR & DEI Analytics (US focus) | Workforce composition, gender/ethnic representation, attrition analysis, compensation equity |
| [Gender Pay Gap Analytics](./gender-pay-gap-analytics/) | Wage Equity Analysis | Gender wage gap, occupation-based disparities, education wage premium, temporal wage trends |
| [HR Employee Analytics](./hr-employee-analytics/) | Workforce Engagement & Retention | Employee representation, attrition rates, engagement metrics, performance correlations, career development |
| [Inclusive Leadership Workshop Analytics](./inclusive-leadership-workshop-analytics/) | Leadership Development Evaluation | Leadership competency assessment, demographic analysis, correlation insights, development priorities |

## Project Structure

Each project follows a consistent structure:

PROJECT_NAME/
├── README.md                 # Project overview, methodology, and insights
├── [dataset].csv             # Source data 
├── analysis.py               # Python script generating analysis and charts
├── charts/                   # Generated visualizations (PNG format)
├── requirements.txt          # Python dependencies
└── [optional] Jupyter notebook or other assets

## How to Use These Projects

1. Explore Individual Projects: Click on any project above to view its detailed README
2. Reproduce Analysis:
   # Navigate to project directory
   cd PROJECT_NAME

   # Install dependencies
   pip install -r requirements.txt

   # Run analysis
   python analysis.py
3. Customize for Organizational Use:
   - Replace the CSV file with your organization's data
   - Update column names in analysis.py to match your data structure
   - Modify visualization parameters to align with corporate branding
   - Adjust demographic categorizations as needed for your workforce
